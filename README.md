# 🌱 Smart Farm Assistant

## 📌 Overview
Smart Farm Assistant is an AI-powered mobile application designed to support farmers in making better agricultural decisions. The app helps users diagnose livestock conditions, choose suitable crops, and get real-time farming advice using artificial intelligence.

---

## 🚜 Problem
Farmers often face challenges such as:

- Lack of quick access to expert agricultural advice  
- Difficulty identifying livestock diseases early  
- Uncertainty in selecting the right crops based on climate and soil  
- Limited digital tools in rural areas  

These issues can lead to productivity loss, disease spread, and financial damage.

---

## 💡 Solution
Smart Farm Assistant provides a simple and accessible solution by integrating AI into farming workflows. It delivers instant recommendations and guidance based on user input.

---

## 🤖 How It Works (AI)
The app uses **Google Gemini AI API** to:

- Analyze livestock symptoms  
- Provide possible diagnoses and risk levels  
- Suggest treatment steps  
- Recommend suitable crops based on environmental inputs  
- Answer user questions through a smart chat assistant  

---

## ✨ Features
- 🐄 Livestock Disease Diagnosis  
- 🌾 Crop Recommendation System  
- 💬 AI Chat Assistant for farming advice  
- 📱 Simple and user-friendly interface  
- 🔐 Firebase Authentication (login/register)  

---

## 🛠️ Tech Stack
- Flutter (Cross-platform development)  
- Firebase (Authentication & Firestore)  
- Google Gemini AI API

- GoRouter (Navigation)
